This is my first day writing blogs!
This idea comes from my online computer programming teacher , he advises us to write blogs to record new ideas and write brief extract of the lessons.And he says it's useful for us when applying for a job , which can let the interviewer know us better.He recommends some Chinese local websites where computer programmers can communicate and write blogs on , however , since I want to practice my English , I decide to write it in English.
Although I haven't learn C# , I built my personal website with the help of github pages and Gridea.It took me some time to download different softwares and struggle with those long and misterious code.But I found a great course on bilibili and it introduces the Gridea , which can help a beginner like me to have a blog.I even have my own site , hahaha!
I think it's really meaningful to develope a software like that , helping those who do not understand computer programming build their blogs.However, I want to DIY one on my own after I learn more about computer programming.
Tommorrow I will learn something about the basic of C# ,so excited to learn more now.
